# Ryoko
Ryoko is a **collaborative trip planning web application** designed for **Gen Z and Millennials** who plan trips together. It centralizes planning, enables **real-time collaboration**, and supports **post-trip community engagement**, reducing the friction of fragmented tools like spreadsheets, group chats, and multiple booking apps.

### Trip Planning
- Create shared trips with destinations, dates, and participants
- Centralized itinerary for activities, transport, and accommodations
- Structured planning to avoid miscommunication and duplicate work

### Real-Time Collaboration
- Live updates using **WebSockets / Socket.IO**
- Multiple users can edit plans simultaneously
- Instant synchronization across all participants

### Group Decision-Making
- Collaborative inputs for activities and plans
- Supports voting, discussion, and consensus-building

### Post-Trip Community 
- Share experiences, tips, and itineraries after trips
- Build a lightweight travel community around shared journeys
